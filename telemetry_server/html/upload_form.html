CalSTAR Telemetry

Upload instructions

The CSV must have headers.

Each data point should have an ID header that increases with increasing time.

Each column should be numeric and correspond to one of the database's accepted datatypes (eg. Time, Altitude, etc.)

Columns that you want the uploader to ignore should have a header of "Ignore"

If the data is not from a new run, a run ID must be entered.
